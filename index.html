Installation de jQuery dans le projet

Deux maière pour le faire

Première maniére

Via un fichier minifier (compressé). Dans le site officiel de 'jQuery' on va copier(Ctrl + a) le fichier compressé fichier minifié de jQuery. Dans le projet en cours ,on va créer un fichier que l'on nomme "jQuery.js" puis aprés on colle( Ctrl + C) le fichier copier depuis le site officiel. Pour relier le ficher 'jQuery', on crée un lien (script src="jQuey") en bas ,et juste avant la balise "body" le fichier "html".

Deuxième maniére

Via CDN .Plusieurs sites (StackPath, Google CDN, microsft CDn, CDNJS CDN, jsDelivr CDN) proposent des URl a insérer directement dans l'attribut "src" de la balise script en bas du fichier HTML .(Plusieurs versions existent : télécharger la plus récente)
